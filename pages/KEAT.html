KEAT-X: Next Generation of the "Knorr Environment for Automated Testing"

Associated with: Knorr-Bremse Technology Center India Pvt. Ltd.

Overview: KEAT-X is a next-generation replacement for the original KEAT system. The maintenance and changes were significantly harder and required more effort in the old KEAT system; KEAT-X was therefore developed from scratch with a new concept and design.

Contributions:

  • Implemented file parser modules in LabVIEW OOPfor XML, XSD, JSON and YAML.
  • Implemented LabVIEW OOP modules and VI scripting for scalable test development.
  • Implemented LabVIEW-based UI to interact with RESTful WebServices for dynamic test generation and reporting.
  • Streamlined version control and issue tracking workflows.

Role in the project:

  • Software Developer

Skills: LabVIEW, VI Scripting, SVN, JIRA, XML Schema Definition (XSD), JSON, Subversion, RESTful WebServices, LabVIEW OOP, YAML, XML

Outcome / Impact: Thoughtful modular code design made tool and the code scalable and maintainable. Addressed challenges from old design.
